# Click v5.8.1 — Multi-Project Edition

Локальное приложение для пакетной публикации в Яндекс.Бизнес.

## Что нового в v5.8.1

### 🏗 Три проекта в одном Click
- **СМУ** (Стальметгрупп) — синий 🏗
- **ИМП** (Инметпром) — зелёный 🔩
- **МПЭ** (МетПромЭнерго) — оранжевый ⚡

Каждый проект:
- свой email Яндекс.Бизнеса
- свой пароль на вход
- свои города (уже заполнены для ИМП и МПЭ)
- свои шаблоны окончаний постов
- своя сессия (cookies изолированы — `users-data/SMU/`, `users-data/IMP/`, `users-data/MPE/`)

## Запуск

### Windows
Двойной клик по `START.bat`.

### Mac/Linux
```bash
./START.command
```

Откроется браузер на `http://localhost:3847`.

## Использование

### Первый запуск
1. На стартовом экране — **3 плитки** проектов
2. Выберите свой проект (СМУ / ИМП / МПЭ)
3. Введите пароль доступа
4. Готово — попадаете в проект уже со всеми городами и шаблонами

### Переключение между проектами
В правом верхнем углу — плашка с активным проектом (с цветом и иконкой). 
Клик → меню → **«↻ Сменить проект»** → возвращаетесь на экран выбора.

## Перенос со старой версии (v5.7.x)

1. Распакуйте `click-v5.8.1.zip` в новую папку
2. Перенесите `node_modules/` из старой версии
3. **СМУ-cookies переносятся автоматически** — при первом входе в СМУ сохранятся в `users-data/SMU/`
4. Запустите `START.bat` → выберите СМУ → пароль `1501`
5. Все ваши сохранённые города и шаблоны СМУ подтянутся из локального хранилища браузера

Для ИМП и МПЭ — 117/137 городов уже зашиты в код, появятся при первом входе.
